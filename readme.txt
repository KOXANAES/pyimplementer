1. Установить python3 
2. Проверка установленного python3 (python3 --version) (Python [версия] если установлен)
3. cd anglina/ (зайти в папку проекта)
4. python3 -m venv venv (создание виртуального окружения)
5. venv\Scripts\activate (активация виртуального окружения) (в терминале будет что-то вроде (venv) user1234:~../angelina$ )
6. pip install -r requirements.txt (установка необходимых пакетов для проекта)
7. python3 server.py (запуск локального сервера)
8. http://localhost:5500 (открыть в браузере - фронтенд приложения)
