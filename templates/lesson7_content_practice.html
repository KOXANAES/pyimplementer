Язык программирования Python

1. Команда вывода

Создайте программу, которая выведет на экран следующий текст: print("Привет, я люблю язык программирования Python!")

2. Вычисление выражения

Создайте программу, которая вычислит значение выражения: result = 10 + 5 * 2

3. Ввод данных и вычисление выражения

Создайте программу, которая вычислит значение выражения, где значение переменных вводится с клавиатуры: a = int(input("Введите a: "))
b = int(input("Введите b: "))
result = a + b

4. Проверка остатка от деления

Создайте программу, которая проверяет, является ли остаток от деления a на b ненулевым. Значение переменных вводится с клавиатуры: a = int(input("Введите a: "))
b = int(input("Введите b: "))
if a % b != 0:
    print("Остаток ненулевой")

5. Интервал времени в секундах

Дан интервал времени в часах, минутах и секундах. Напишите программу, которая определит тот же интервал в секундах: hours = int(input("Введите часы: "))
minutes = int(input("Введите минуты: "))
seconds = int(input("Введите секунды: "))
total_seconds = hours * 3600 + minutes * 60 + seconds
