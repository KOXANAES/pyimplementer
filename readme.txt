1. cd pyimplemener (зайти в папку проекта)
2. Удалить папку venv, если таковая имеется
2. python3 -m venv venv (создание нового виртуального окружения)
5. venv\Scripts\activate (активация виртуального окружения) (для Linux - source venv/bin/activate)
6. pip install -r requirements.txt (установка необходимых пакетов для проекта)
7. python3 server.py (запуск локального сервера)
8. http://localhost:5500 (открыть в браузере - фронтенд приложения)
